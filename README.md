# weather-django
A weather app using django with api of openweathermap

You can search weather forecast for 5 days with data every 3 hours by location, city name.



# To run go to Core => views.py => In line number 5 put your API key. Or <a href="[https://openweathermap.org/](https://github.com/kafle-kalyan1/weather-django/blob/main/core/views.py)">Here</a>
